# SkillScout AI

Auto-generated by ETS-AI 10x.

## Idea Summary
An AI-powered resume screener for Indian fresher software engineers to help them apply for jobs.

## Problem Statement
Indian fresher software engineers struggle to tailor their resumes effectively for job applications, leading to missed opportunities due to mismatches with job descriptions and lack of identified missing skills.

## Target Persona
Indian fresher software engineer

## MVP Features
- User account creation and management (sign-up, login)
- Resume upload (PDF, DOCX) and parsing
- Job description input (text paste or URL)
- AI-powered resume-job description matching and compatibility scoring
- Specific keyword suggestions for resume optimization
- Identification of critical skill gaps based on job description
- Tailoring recommendations for existing resume sections (e.g., projects, experience)
- Interactive editor to apply suggestions and view real-time score updates
- Download/export tailored resume
- Dashboard to view past analyses and saved resumes/job descriptions

## Tech Stack
Python for backend logic and ML (FastAPI/Django), React/Vue.js for frontend, PostgreSQL for database, AWS/GCP for cloud infrastructure, Hugging Face/OpenAI APIs for advanced NLP if needed.
